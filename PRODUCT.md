# Product

## Register

product

## Users

Hobbyists and curious learners exploring the method of loci for fun. They build isometric memory museums — placing symbolic objects in themed rooms — as a creative exercise rather than a high-stakes study tool. Context is relaxed, exploratory, personal.

## Product Purpose

A visual memory museum builder that lets users create, furnish, and walk through isometric rooms tied to memories. Success is when building a museum feels like curating a personal world, not using a productivity app.

## Brand Personality

Warm, Scholarly, Elegant. The feel of browsing a private antiquarian collection — unhurried, rich in texture, quietly impressive. Not performatively old; genuinely considered.

## Anti-references

- Generic SaaS / Notion-style flat tool UI — no corporate blandness
- Gamified / Duolingo — no childish rewards, bright cartoon aesthetic, or dopamine loops
- Dark terminal / hacker — no cold monospace dev-tool energy
- Anything that reads as AI-generated template output

## Design Principles

1. **Atmosphere over chrome** — the museum is the star; UI recedes into the room's mood
2. **Tactile warmth** — surfaces should feel material (wood, leather, paper) not digital
3. **Quiet confidence** — elegant without trying hard; no decoration for decoration's sake
4. **Explore, don't optimize** — the interface invites lingering, not speed-running
5. **Every theme is a world** — switching themes should feel like entering a different building, not toggling a color variable

## Accessibility & Inclusion

- WCAG AA minimum (4.5:1 body text, 3:1 large text and UI controls)
- Respect prefers-reduced-motion
- Color should never be the sole indicator of state
